# Maps App TypeScript
A simple TypeScript app that generates a random user and company, and put their location with clickable markers on the Google Map.

## What's inside
1. **TypeScript**
2. **Parcel Bundler**
3. **Faker**
4. **Google Maps**

## Getting started
1. **Install**
```shell
   npm install

```
2. **Run the local bundler**
```shell
   npm start

```

3. **Open the browser at `http://localhost:1234`**
